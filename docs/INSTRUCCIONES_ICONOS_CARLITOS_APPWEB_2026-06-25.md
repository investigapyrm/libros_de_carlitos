# Instrucciones para generar iconos de Carlitos para la appweb

Fecha: 2026-06-25
Proyecto: Carlitos y la Economia Ambiental Sostenible
Uso previsto: appweb educativa, botones, tarjetas, misiones verdes, indicadores, estados y futuras piezas de difusion

## Advertencia editorial y legal

Antes de generar o publicar iconos con la figura real de Carlitos, confirmar autorizacion de los autores o titulares del personaje y de los materiales fuente.

Si todavia no existe autorizacion formal, usar este modo seguro:

```text
Crear iconos con un yacare infantil original, inspirado en fauna paraguaya, sin copiar exactamente la figura existente de Carlitos, pero manteniendo un tono compatible con una appweb educativa ambiental.
```

No incluir logos institucionales, marcas comerciales, banderas, escudos ni texto legible dentro de los iconos salvo autorizacion expresa.

## Objetivo visual

Crear al menos 20 iconos consistentes, alegres y muy atractivos para ninos, utiles en la appweb "Carlitos y los residuos que se convierten en oportunidad".

Los iconos deben funcionar en:

* botones principales;
* tarjetas de capitulos;
* misiones verdes;
* indicadores;
* controles de lectura;
* estados online/offline;
* secciones de actividad;
* futuras versiones PWA.

## Estilo recomendado

```text
Iconos infantiles editoriales tipo sticker premium, limpios, expresivos y muy claros. Estilo coherente con una appweb educativa ambiental de Paraguay.

Forma: iconos centrados, silueta simple, bordes redondeados, volumen suave, detalles suficientes para ser atractivos sin perder legibilidad.

Color: verdes naturales, cielo suave, tierra clara, blanco, acento dorado, pequenos acentos azul/amarillo cuando representen separacion de residuos.

Tono: alegre, pedagogico, comunitario, moderno, institucional y apto para ninos.

Fondo: transparente si la herramienta lo permite. Si no permite transparencia, usar fondo blanco puro o verde muy claro para recorte posterior.

No incluir texto, letras, numeros, logos, marcas, sellos, banderas ni simbolos institucionales.

Formato final deseado: PNG 1024x1024, icono centrado, margen seguro de 12%, alta resolucion.
```

## Prompt maestro para GPT Imagenes

Usar este prompt al iniciar la serie:

```text
Vamos a crear una serie de iconos para una appweb educativa infantil titulada "Carlitos y los residuos que se convierten en oportunidad".

Necesito que todos los iconos mantengan el mismo estilo visual, la misma paleta y el mismo nivel de detalle.

El universo visual pertenece a una historia educativa ambiental en Paraguay, con un yacare nino amigable como personaje conductor. Los iconos deben sentirse alegres, claros y pedagogicos, aptos para ninos, docentes y familias.

Estilo: icono infantil editorial tipo sticker premium, bordes suaves, volumen sutil, colores naturales, acabado limpio, sin ruido visual, sin texto, sin logos, sin marcas. Deben verse bien en tamano pequeno dentro de una appweb y tambien en tarjetas grandes.

Paleta: verde natural, verde institucional, cielo suave, tierra clara, blanco, acento dorado, pequenos acentos azul y amarillo solo cuando ayuden a distinguir residuos o estados.

Fondo: transparente si es posible. Si no, fondo blanco puro o verde muy claro, facil de recortar.

Composicion: icono centrado, margen seguro, sin elementos cortados, forma reconocible de inmediato.

Voy a pedir una serie de iconos. Mantener consistencia total entre todos.
```

## Flujo recomendado de generacion

1. Generar primero una hoja de estilo con 8 iconos de prueba.
2. Elegir el estilo mas claro y atractivo.
3. Usar ese resultado como referencia visual para crear los 24 iconos finales.
4. Generar cada icono individual en PNG 1024x1024.
5. Revisar que no aparezcan textos raros, logos inventados ni marcas.
6. Exportar versiones comprimidas para web si se publican.

## Variantes utiles por icono

Para cada icono final, pedir si es posible:

```text
Genera 3 variantes del mismo icono:
1. version completa tipo sticker con sombra suave;
2. version plana sin sombra para botones pequenos;
3. version monocromatica verde para estados secundarios.

Mantener el mismo encuadre, sin texto, sin logos, fondo transparente si es posible.
```

## Control de calidad

* El icono se entiende en tamano pequeno.
* No incluye texto, letras, numeros, logos ni marcas.
* No hay simbolos institucionales inventados.
* No hay residuos desagradables ni escenas de riesgo.
* No hay pobreza caricaturizada ni representaciones indignas.
* El estilo coincide con los demas iconos.
* El personaje Carlitos no se usa sin autorizacion formal.
* El archivo queda en PNG cuadrado con margen suficiente.
* El fondo es transparente o facil de recortar.
* La paleta no se vuelve monotona; combina verde, tierra, cielo, blanco y acento dorado.

## Recomendacion para integracion web

Antes de publicar:

* comprimir los PNG;
* crear versiones `webp` si el flujo de despliegue lo permite;
* guardar originales en `assets/icons/source/`;
* usar nombres estables;
* probar contraste sobre fondos claros y oscuros;
* validar en celular que los iconos no se vean borrosos.
